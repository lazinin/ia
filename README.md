# Task Manager

Автор: Лазинин Павел

## Описание программы
Task Manager - это консольное приложение для управления списком задач. 

В приложении реализованы:
- Инкапсуляция, наследование (подкласс UrgentTask) и полиморфизм.
- Структуры данных: Очередь (Queue) для приоритизации задач и Стек (Stack) для паттерна "Отмена действия".
- Сохранение и загрузка состояния базы данных в формате JSON.
- Валидация пользовательского ввода и строгая фильтрация по статусам и приоритетам.

## Как запустить приложение
1. Убедитесь, что у вас установлен Python (версии 3.7 или выше).
2. Склонируйте репозиторий:
    ```bash
    git clone "https://github.com/lazinin/ia.git"
    ```

3. Перейдите в директорию проекта.
4. Запустите файл:
   ```bash
   python main.py
   ```

## Примеры использования
* Добавление задачи: выберите пункт меню 2, введите название, описание (опционально) и статусы
* Отмена действия: Добавьте задачу, затем выберите пункт 8. Последняя добавленная задача исчезнет
* Очередь по приоритетам: Создайте задачи с приоритетами Low и High. Выберите пункт 7 — система выведет задачи, отсортированные по важности (демонстрация очереди)
